# amv-system-demo-app

## Build & development
```bash
npm install
grunt
```

Run `grunt` for building and `grunt serve` for preview.

### Testing

Running `grunt test` will run the unit tests with karma.

## About
The AMV System Demo is a work-in-progress proof-of-concept prototype application.
